W3C Open Source Software - News for 2004

This is an archive of the W3C open source software release news for the year 2004. Please read the Latest News for up-to-date information

HTML-XML-utils 3.6

20 December 2004: version 3.6 of the HTML-XML-utils adds a new program, name2id, that converts old-style anchors to new style ones . See the Changelog for changes in earlier versions.

W3C Link Checker 4.1

25 November 2004: A new version of the W3C link checker fixes a few bugs and slightly improves support for the robots exclusion protocol. read the release notes for details on downloading the tool or Use it online.

W3C Log Validator 0.5

18 Novembre 2004: A new version of the Log Validator, the free web server log analysis tool with a focus on the quality of Web documents. Some bug fixes, better API documentation and a few additional features.

HTML-XML-utils 3.5

8 November 2004: version 3.5 of the HTML-XML-utils corrects a regression in the the man pages for incl and addid. See the Changelog for corrections in earlier versions.

Amaya 8.7

21 October 2004: Amaya 8.7 is released. This release fixes a lot of reported bugs and includes some new features. See the list of bug fixes/new features ( http://www.w3.org/Amaya/User/New) for more details.
